Timestamp Microservice

Either enter the date in the input box above, or pass it as parameter to url. You can enter date in natural language format or unix timestamp.

Example

Input:

https://timestampmicroserviceapi.herokuapp.com/11%20mar%2093
https://timestampmicroserviceapi.herokuapp.com/731808000

Output:

{ "natural": "March 11, 1993", "unix": 731808000 }
